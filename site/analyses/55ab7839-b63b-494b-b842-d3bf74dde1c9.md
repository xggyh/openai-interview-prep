## 题目本质

**LC 3453 Separate Squares I**：给一组轴对齐正方形 `squares[i] = [x, y, side]`（左下角 + 边长）。找一条**水平线** y = L 使**所有正方形被该线切割的上下面积**相等。返回 L。

**二分实数**经典题。

## 解法

定义 `f(L)` = 所有 square 中 y < L 部分的总面积。`f(L)` 单调不减。要找 `f(L) = total / 2` 的 L。

二分 L ∈ [min_y, max_y_top]，求满足条件的最小 L。

## Python 实现

```python
from typing import List

class Solution:
    def separateSquares(self, squares: List[List[int]]) -> float:
        total = sum(side * side for _, _, side in squares)
        target = total / 2

        def below(L: float) -> float:
            """所有 square 的 y < L 部分总面积"""
            s = 0
            for x, y, side in squares:
                if L <= y:
                    pass   # 整体在上方
                elif L >= y + side:
                    s += side * side   # 整体在下方
                else:
                    s += (L - y) * side
            return s

        lo = min(y for _, y, _ in squares)
        hi = max(y + side for _, y, side in squares)
        # 二分到精度 1e-9
        for _ in range(60):  # 2^60 步长够精度
            mid = (lo + hi) / 2
            if below(mid) < target:
                lo = mid
            else:
                hi = mid
        return lo
```

## 复杂度

- 时间：**O(N × 60)** = O(N)，60 次二分（精度 2^(-60)）
- 空间：O(1)

## 关键技术点

### 1. f(L) 单调不减

L 增大 → 每 square 的下半部分 ≥ 原来。所以 f 是非降的函数，**二分必然找到**。

### 2. 二分实数精度

不要 `while lo < hi` —— 浮点比较不可靠。**固定迭代次数**（如 60-100 次）保证精度。

### 3. 三段 case

- L ≤ y：square 全在上方
- L ≥ y + side：全在下方
- 否则：下半 = (L - y) × side

### 4. 范围

`lo = min_y`（再低没意义，下面是 0），`hi = max(y + side)`（再高所有 square 都在下方，f 已饱和）。

## 易错点

> [!pitfall]
> ❌ 用整数二分 —— 答案可能非整；
> ❌ 浮点 while 终止条件用 `<` 而非"固定迭代次数"；
> ❌ f(L) 三段判断写错 —— 边界 L == y 或 L == y+side；
> ❌ 用 `==` 比较浮点 —— 永远 False，注意。

> [!key]
> 二分**实数** + 单调函数 = 求方程解。同模板：LC 410 Split Array Largest Sum（二分答案）、LC 668 Kth Smallest Number in Multiplication Table。

> [!followup]
> "垂直线代替水平线？" → 对称处理 x；"任意角度线？" → 复杂得多（旋转 + 二分）；"圆而非正方形？" → 积分计算下半圆面积，仍 monotonic 二分可解；"返回所有合法 L？" → 解集是单点（除非有水平边重合）。
